# Проект онлайн кинотеатра «Что посмотреть»

Установка зависимостей `npm i`

Запуск в dev режиме `npm start`

Запуск в prod режиме `npm run build`

Запуск тестов `npm run test`

[Live demo](https://wtw-online.vercel.app/)
